üßπ Detoxify
unitaryai/detoxify
A single model that returns six independent toxicity scores in one forward pass ‚Äî far more informative than a binary hate/non-hate label. Built on RoBERTa-base and trained on the Jigsaw Toxic Comment dataset (~1.8M Wikipedia comments). Replaces the previous HateBERT + SarcasmNet dual-model setup with a lighter, faster, and more granular alternative.
1Text is tokenised by RoBERTa's BPE tokenizer (max 512 tokens) and passed through 12 transformer layers.
2A multi-label classification head outputs six sigmoid scores simultaneously ‚Äî each score is independent, so a comment can score high on multiple categories at once.
3Severity is derived from the toxicity score: >65% ‚Üí high, >45% ‚Üí medium, >30% ‚Üí low. All six scores are logged and displayed in the playground.
4The identity_attack and threat scores act as secondary signals ‚Äî if either exceeds 0.50 the severity is promoted one level regardless of the main toxicity score.
